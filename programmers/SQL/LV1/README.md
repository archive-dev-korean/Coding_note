***암기사항***
- 비트 연산자 (효율적인 수 처리, 집합 표현, 상태 압축 등에서 사용)
  > &(and) 둘다 일이면 일
  > 
  >|(OR) 하나라도 1이면 1
  > 
  >^(XOR) 다를 떄 1
  > 
  > ~(NOT) 비트 반전(보수)
  > 
  >  `<<`(왼쪽 쉬프트 2의 곱)
  > 
  >  `>>`(오른쪾 쉬프트 2의 나눗셈)
  >
  > ex) select 6&3;  ---- 110 & 011 = 010 ->2
  > 
  > ex) select 6|3;  ---- 110 | 011 = 111 ->7
