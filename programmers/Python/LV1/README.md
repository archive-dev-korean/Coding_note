***2차원 배열에서 층과 열 구하기***
- 번호(num)주어 졌을 떄:</br>
- w : 한 층의 개수 제한</br>
> 1.층(row) = (num -1) // w </br>

- 2.열(index) 계산</br>
>방향이 지그재그 일때: (층마다 다르다고 생각해야함)
>
```
if row % 2 == 0:  # 짝수층이면 왼→오
    col = (num - 1) % w
else:             # 홀수층이면 오→왼
    col = (w - 1) - ((num - 1) % w)
```
정리 : 층마다 방향이 다를 때
| 계산 항목  | 공식                                                        |
| ------ | --------------------------------------------------------- |
| 층(row) | `(num - 1) // w`                                          |
| 열(col) | `짝수층: (num - 1) % w`<br> `홀수층: (w - 1) - ((num - 1) % w)` |
